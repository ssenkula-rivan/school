How to Import Data
Method 1: File Upload (Recommended)
  1. Download the appropriate template
  2. Fill in your data following the template format
  3. Save as CSV or Excel file
  4. Upload the file using the import buttons above
Method 2: Database Connection
  1. Get database credentials from your IT team
  2. Click "Connect to Database"
  3. Enter connection details
  4. Test connection and import data
